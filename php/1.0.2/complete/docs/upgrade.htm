Upgrading

A guide to upgrading from a previous version.

From 1.0.0
Backup your files and data. Now grab the changed files from the website and upload them, overwriting the existing files. Run install/upgrade/from_1.0.0.php and this will make all the required SQL changes. Finally delete the install folder and also version.php as it is no longer used or included. Once you've done that, you're done!
