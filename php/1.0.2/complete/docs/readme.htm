Installation
Installing Particle Whois is pretty simple. This section will guide you through the steps to have your script up and running in minutes.

Back to Top

Database configuration
In order to install Particle Whois you will need to create a MySQL database and a user which has permissions to access this database.

Once you have these, open up includes/config.php. Here you will need to enter your information. Usually you should be able to leave $dbhost and $dbprefix as default. You don't need to change these unless you know what your doing.

You will however need to change:

$dbuser = "";
$dbpass = "";
$dbname = "";

Enter the name of your database into $dbname such as $dbname = "mydatabase";. You also need to enter the username for the user who can access this database into $dbuser and the password for that user into $dbpass.

Once you have entered these values, save config.php and upload it to your web server. You may want to remove write permissions to it so that you do not overwrite it in future (although make sure it still has read permissions so the script can read it!).

Back to Top

Running the installer
Once you have uploaded the file you can access the installer and run the script:

install/install.php

If you are reading this on the website you will be able to click the above link but if not simply type the address into your web browser. The installer will then take you through each step.

Back to Top

Delete /install/
For security reasons, once you have run the installer you will need to delete the install folder from the web server. The rest of the script will not be accessable until you have done this.

Admin Panel
You can reach the admin panel by goign to admin.php from your homepage. You will then be asked to log in using the username and password you choosen when installing the script.

Back to Top

History Log
This allows you to view the history of whois searches, each domain is listed along with it's availability at the time of the search and the date and time at which the search was run. You can remove old searches using the prune log feature. Enter an amount of days (the default is 30) and click prune to have all searches run before the specified amount of days removed.

Back to Top

Changing Settings
Particle Whois has several options for customisation. To make changes to these settigns, log in to the admin panel and click change settings. You will then have access to the change settings menu, each one with an explanation of what it does.
